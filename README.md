## AlgoTrading
-------------------------

基于Python的多资产，基于事件循环的量化交易策略回测框架。

## 依赖
-------------

见requirements目录；
除此之外，还需要安装Finance-Python:
https://github.com/ChinaQuants/Finance-Python
以及tushare：

~~~
pip install tushare
~~~

pandas_datareader:

~~~
pip install pandas_datareader
~~~

以及pymssql

~~~
pip install pymssql
~~~
